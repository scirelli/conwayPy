# conwayPy
Conway's game of life in Python.

Requires python 3.

## Installation
Install [pygame](http://www.pygame.org/wiki/GettingStarted#Pygame%20Installation)

`python3 -m pip install pygame --user`

### Graphics version

`python3 conwayGraphics.py`

### Text version

`python3 conwayText.py`
